# Desarrollo efectivo de aplicaciones con mono-repositorios
Mono-repositorios o *Mono-repository* como se les conoce en inglés es una estrátegia de gestión del control de versiones de código fuente que consiste en agurpar en un unico repositorio de código todos los componentes de la solución.
